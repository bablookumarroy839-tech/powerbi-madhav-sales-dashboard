# 📊 Madhav E-Commerce Sales Dashboard

## Overview

This project presents an interactive Sales Analytics Dashboard developed using Power BI to analyze sales performance, profitability, customer behavior, and product trends.

## Objectives

- Analyze overall sales and profit performance.
- Identify top-performing customers and states.
- Monitor monthly profit trends.
- Understand customer payment preferences.
- Support data-driven business decisions.

## Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Total Quantity Sold
- Average Order Value (AOV)
- Profit Margin
- Top Customer
- Top State
- Best Selling Category

## Dashboard Features

- Interactive Filters
- Sales Analysis by State
- Customer-wise Analysis
- Category-wise Performance
- Monthly Profit Trend
- Payment Mode Analysis
- Dynamic KPI Cards

## Tools & Technologies

- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization
